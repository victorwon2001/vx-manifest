# tamp스크립트 작업 규칙

- 한국어로 작업 로그와 문서를 작성한다.
- 이벗매니저 접근이 필요한데 로그아웃 상태면 먼저 로그인 후 사용자가 요구한 사이트나 접근권한이 필요한 곳으로 진입한다.
- 스크립트 추가/개선 시에는 `tools/release.ps1`로 버전업, 한국어 changelog, git commit, git push를 함께 수행한다.
- 로더는 공통 진입점만 유지하고, 사이트별 기능은 `scripts/<scriptId>/main.js`로 분리한다.
- 각 PC의 스크립트 ON/OFF는 로더의 로컬 저장소를 사용한다.
